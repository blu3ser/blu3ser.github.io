EW Test

In the following charts, I examine how three EW aircraft (EA-18G Growler, EC-37B Compass Call, and J-16D) impact the detection of an F-16A aircraft across different radar systems. The negative distance values represent situations where no EW aircraft is present. We conducted tests using CMO version 1328.5 an coarse simulation (flame speed), evaluating distances from 0 to 400 nautical miles.
